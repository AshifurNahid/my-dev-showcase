## 🚀 Quick Start

### Prerequisites
- Node.js 16+ and npm/yarn/pnpm

### Installation

```bash
# Clone the repository
git clone https://github.com/AshifurNahid/my-dev-showcase.git
cd my-dev-showcase

# Install dependencies
npm install
# or
yarn install
# or
pnpm install

# Start development server
npm run dev

# Build for production
npm run build
```

The development server will start at `http://localhost:5173`
